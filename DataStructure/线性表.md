# 线性表

## 定义

线性表是具有相同特性的数据元素的一个有限序列

## 逻辑特征

- 在一个非空的线性表中，有且仅有一个**开始节点**，它没有直接前趋，而只有一个直接后继
- 在一个非空的线性表中，有且仅有一个**终端节点**，它没有直接后继，而只有一个直接前趋
- 线性表的**内部结点**(节点个数 >= 3)都有且仅有一个直接前趋和一个直接后继

## 抽象数据类型

```
ADT List {
    数据对象: D = {a¡ | a¡属于Elemset, (i = 1,2,...,n, n≥0)}
    数据关系: 序偶关系
    基本操作:
        InitList(&L)
            操作结果: 构造一个空的线性表
        DestroyList(&L)
            初始条件: 线性表L已存在
            操作结果: 销毁线性表L
        ClearList(&L)
            初始条件: 线性表L已存在
            操作结果: 将线性表置为空表

        ListEmpty(L)
            初始条件: 线性表L已存在
            操作结果: 若线性表L为空表,则返回TRUE,否则返回FALSE
        ListLength(L)
            初始条件: 线性表L已存在
            操作结果: 返回线性表L中的数据元素个数

        GetElem(L,i,&e)
            初始条件: 线性表L已存在, 1<=i<=ListLength(L)
            操作结果: 用e返回线性表L中第i个元素的值
        LocateElem(L,e,compare())
            初始条件: 线性表L已存在, compare()是数据元素判定函数
            操作结果: 返回线性表L中第一个与e满足compare()的数据元素的位序,若这样的元素不存在则返回值为0

        PriorElem(L,cur_e,&pre_e)
            初始条件: 线性表L已存在
            操作结果: 若cur_e是L的数据元素且不是第一个,则使用pre_e返回它的前趋;否则操作失败,pre_e无意义
        NextElem(L,cur_e,&next_e)
            初始条件: 线性表L已存在
            操作结果: 若cur_e是L的数据元素且不是最后一个,则使用next_e返回它的后继;否则 操作失败,next_e无意义

        ListInsert(&L,i,e)
            初始条件: 线性表L已存在, 1<=i<=ListLength(L)+1
            操作结果: 在L的第i个位置之前插入新的数据元素e, L的长度加一
        ListDelete(&L,i,&e)
            初始条件: 线性表L已存在, 1<=i<=ListLength(L)
            操作结果: 删除L的第i个数据元素, 并用e返回其值, L的长度减一
        ListTraverse(&L, visited())
            初始条件: 线性表L已存在
            操作结果: 依次对线性表的每个元素调用visited()
} ADT List

```
